# dwarf(special judge)

일곱 난장이가 저녁 시간에 9 명이 와서는 모두 자기가 7 명 중에 한명이라고 우기고 있다.

난장이들은 모자를 쓰고 있고 모자에는 100 보다 작은 수가 쓰여져 있다.

다행스럽게도 일곱난장이들은 이 번호의 합이 100 이라는 것을 당신은 안다. 일곱 난장이들의 번호를 골라라.

입력
100 보다 작은 9 개의 수가 입력으로 주어진다.
출력
7 개의 수를 출력한다. 순서는 관계가 없다.
